# Billie Eilish — Tribute

An immersive, animation-driven single-page tribute to Billie Eilish, built as a
front-end **portfolio centerpiece**. Cinematic, editorial, and kinetic.

> Unofficial fan-made portfolio project. Not affiliated with the artist.

## Tech stack

- **React 18 + TypeScript + Vite**
- **Tailwind CSS v4**
- **GSAP + ScrollTrigger** — pinned sections, horizontal scroll, parallax
- **Lenis** — momentum smooth scrolling
- **Motion** (Framer Motion) — gestures, reveals, layout animations
- **lucide-react** — icons

## Highlights

- Animated **preloader** with a counter + name reveal
- **Custom cursor** and **magnetic** nav / buttons
- **Lenis smooth scroll** synced with GSAP ScrollTrigger
- Top **scroll-progress bar** + working fullscreen **mobile menu**
- **Hero** with parallax performance photo
- **About** with a scroll-pinned word-reveal + stats
- **Discography** as a scroll-driven **horizontal pin** gallery
- Embedded **Spotify player** with real tracks
- Animated **tour list**, **clip-path** reveals, and masonry **gallery**

## Running the code

```bash
npm install     # install dependencies
npm run dev     # start the dev server (http://localhost:5173)
npm run build   # production build into /dist
```
